# tool-containers

Docker images for many purposes — AI agents, CI builders, sandboxes, and more. Each image is a self-contained "thing" inside a category, with multiple build variants, per-platform deployment recipes, and its own CI pushing to GHCR, Docker Hub, and Quay.

Repo conventions live in [`.structure/`](./.structure/structure.md).
